CSS Position Property Guide

Static (default)

This box uses position: static. It flows naturally.
HTML & CSS Code:
HTML
CSS
<div style="position: static; background-color: #f8f9fa; padding: 15px; border-radius: 6px; border-left: 4px solid #6c757d;">
  This box uses <strong>position: static</strong>. It flows naturally.
</div>
/* Static is the default position value */
div {
  position: static;  /* Can be omitted as it's default */
  background-color: #f8f9fa;
  padding: 15px;
  border-radius: 6px;
  border-left: 4px solid #6c757d;
}

Relative

This box uses position: relative. It's moved 20px down and right.
HTML & CSS Code:
HTML
CSS
<div style="position: relative; top: 20px; left: 20px; background-color: #e3f2fd; padding: 15px; border-radius: 6px; border-left: 4px solid #2196f3;">
  This box uses <strong>position: relative</strong>. It's moved 20px down and right.
</div>
div {
  position: relative;
  top: 20px;      /* Moves element 20px from its top */
  left: 20px;     /* Moves element 20px from its left */
  background-color: #e3f2fd;
  padding: 15px;
  border-radius: 6px;
  border-left: 4px solid #2196f3;
}

Absolute

This box uses position: absolute inside a relative container.

This section is the relative container for the absolute box.

HTML & CSS Code:
HTML
CSS
<section style="position: relative;">
  <div style="position: absolute; top: 10px; right: 10px;">
    This box uses position: absolute
  </div>
</section>
section {
  position: relative;  /* Creates positioning context */
}

div {
  position: absolute;
  top: 10px;      /* Positions from top of relative parent */
  right: 10px;    /* Positions from right of relative parent */
}

Sticky

I stick to the top when scrolling! (position: sticky)
Scroll me! I'm long content so you can see sticky behavior.
HTML & CSS Code:
HTML
CSS
<div style="height: 300px; overflow-y: scroll;">
  <div style="position: sticky; top: 0;">
    I stick to the top when scrolling!
  </div>
  <div style="height: 800px;">
    Scroll me!
  </div>
</div>
/* Container */
div {
  height: 300px;
  overflow-y: scroll;
}

/* Sticky element */
div.sticky {
  position: sticky;
  top: 0;         /* Sticks when it reaches top */
}

/* Content to scroll */
div.content {
  height: 800px;  /* Long content to enable scrolling */
}
I'm a fixed box! I stay in place when you scroll.
Fixed Position Code:
HTML
CSS
<div style="position: fixed; bottom: 20px; right: 20px;">
  I'm a fixed box!
</div>
div {
  position: fixed;
  bottom: 20px;   /* Distance from bottom */
  right: 20px;    /* Distance from right */
  /* Other styles */
}
